# Software-Website-Clone
I stumbled on this website while checking out relatively easy websites to clone with just HTML and CSS. This repo contains my clone of the said software website. 
# Website Link.
https://tadederhbi.github.io/Software-Website-Clone/ The provided link shows the live view of the page. 
# DAYS USED. 
A little under one week as a result of power outage during the third and fifth day of coding. 
# LANGUAGES USED: 
As expected, the major part of this code was written in HTML to show the layers of the the web page as would be displayed on the deployed GitHub Live Page. Then, i made use of the usual CSS as we know it. 
At some point, i decided to make use of a CSS library to allow ease of materials for buiilding the site. 
I found a chrome extention named "Font Details" which proved instrumental in not only providing the font family name, but went further in showing the size, the line spacing, word spacing, as well as font color. 
Making use of Font Details, with the help of Bootstrap Icons, and a number of Bootstrap components. 
I am expected to add a bit of functionality to the website but that part is unatended to at this moment. I would make sure to edit this README.md file when i make that part operational.
